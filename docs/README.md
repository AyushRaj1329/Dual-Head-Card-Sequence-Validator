# Documentation

This folder contains documentation assets for the Card Sequence Validator project.

## Folder Structure

### `/screenshots/`
Place your application screenshots here with the following naming convention:
- `main-dashboard.png` - Main application dashboard
- `network-setup.png` - Network configuration interface
- `file-management.png` - File management window
- `live-scanning.png` - Real-time validation logging
- `video-thumbnail.png` - Thumbnail for demo video

### `/demo/`
Place your demo videos here:
- `card-validator-demo.mp4` - Main application demonstration video

## Screenshot Guidelines

### Recommended Screenshots:
1. **Main Dashboard** - Show the home page with both heads' status
2. **Network Configuration** - Display the dual-head network setup
3. **File Management** - Show file loading and preview functionality
4. **Live Scanning** - Capture real-time validation in action
5. **Card Type Selection** - Show the card type selector dialog
6. **Validation Results** - Display successful validation logs

### Image Specifications:
- **Format**: PNG (preferred) or JPG
- **Resolution**: 1920x1080 or higher
- **Quality**: High quality, clear text
- **Theme**: Use dark theme for consistency
- **Content**: Remove any sensitive data or real card numbers

### Demo Video Guidelines:
- **Duration**: 2-5 minutes
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 minimum
- **Content**: Show complete workflow from startup to validation
- **Audio**: Optional narration explaining features

## Usage in README

The main README.md file references these assets:
- Screenshots are embedded using relative paths
- Demo video is linked with a thumbnail
- All paths are relative to the repository root

Update the README.md file paths if you change the file names or structure.
